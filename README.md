# Vision AI Studio

Minimal Vite setup for deploying on Vercel.

## Local development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel settings

Use these values in Vercel:

- Framework Preset: **Vite**
- Root Directory: **./**
- Install Command: **npm install**
- Build Command: **npm run build**
- Output Directory: **dist**
